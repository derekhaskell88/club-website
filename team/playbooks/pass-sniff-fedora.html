 Password Sniffing fedora 

  • Cmd: which passwd | Response: (/usr/bin/passwd)
  • Cmd: cd | Response: [username@servername "]$
  • Cmd: mkdir stakeholders; cd stakeholders | Response: [username@servername stakeholders]
  • Cmd: cp /usr/bin/passwd passwd.bak
  • Cmd: sudo dnf reinstall passwd
  • Cmd: passwd (#change your password)
  • If app is needed to restore from backup
    • Cmd: sudo cp ~/stakeholders/passwd.bak /usr/bin/passwd
